# Mobile Game Project

A simple mobile-friendly HTML5 game built with vanilla JavaScript. The game features touch controls and responsive design, making it perfect for mobile devices.

## Game Features
- Touch and mouse controls
- Responsive canvas that adapts to screen size
- Score tracking system
- Obstacle avoidance gameplay
- Mobile-optimized interface

## How to Play
1. Tap the Start Game button to begin
2. Control the green square by touching/dragging on the screen
3. Avoid the falling red obstacles
4. Score points for each obstacle you successfully avoid
5. Try to achieve the highest score possible!

## Project Structure
- `index.html` - Main game page
- `css/style.css` - Game styling and mobile optimization
- `js/game.js` - Game logic and mechanics

## Mobile App Conversion
This project is designed to be easily converted into a mobile app using WebIntoApp.com.
